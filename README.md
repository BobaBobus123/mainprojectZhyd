# ExpenseTracker Pro

Персональний менеджер витрат на C++/Qt6 з візуалізацією даних.

## Функціональність
- Збереження витрат у локальну БД SQLite.
- Автоматична валідація введених сум.
- Візуалізація статистики через кругову діаграму (Qt Charts).
- Адаптивний інтерфейс (Layouts & Splitters).

## Інструкція зі збірки
1. Встановіть Qt 6.10+ (MinGW).
2. Відкрийте `CMakeLists.txt` у Qt Creator.
3. Виконайте `Run CMake`.
4. Запустіть проєкт (Ctrl+R).

## Сценарій перевірки
1. Натисніть кнопку "Додати витрату".
2. Введіть категорію (напр. "Їжа") та суму.
3. Переконайтеся, що запис з'явився в таблиці, а діаграма оновилася.
4. Спробуйте видалити запис через контекстне меню або кнопку видалення.
